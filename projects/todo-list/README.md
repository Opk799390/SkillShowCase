# Todo List (Demo)

This is a small vanilla JavaScript Todo List demo included in the portfolio.

Features:
- Add, edit, delete todos
- Mark complete
- Persisted to localStorage

How to run locally:
1. Open `projects/todo-list/index.html` directly in your browser, or run a static server (recommended):

```bash
# macOS / Linux
python3 -m http.server 8000
# then open http://localhost:8000/projects/todo-list/
```

Notes:
- No build step required.
